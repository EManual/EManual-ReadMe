编程助手开发文档 [![Build Status](https://travis-ci.org/EManual/docs.svg)](https://travis-ci.org/EManual/docs)
-------------

在你参与EManual开发、学习前，请查看本文档，包含一些规范和约定。

### 安装

文档生成工具[docpress](https://github.com/docpress/docpress)
```
$ npm install -g docpress
```

### 本地预览

```
$ make serve
```

### 提交修改并推送到GitHub

```
$ make syncUpdate
```
