## 用户反馈模块

保存记录

```json
{
    "content":"{反馈内容}"  
    ,"type":"report|advice" //report:报告文章错误等 advice:反馈建议
    ,"status":"open|close" //反馈状态
    ,"app_version":"2.0.0" //应用版本
    ,"system_version":"4.4.4" //用户操作系统版本
    ,"model" :"MX 4"       //用户手机品牌
    ,"user_contact" : "foo@bar.com" //用户联系方式,一般是邮箱
    ,"createAt":"{自动生成,忽略}" //创建时间
    ,"updateAt":"{自动生成,忽略}" //更新时间
}

```
